# Website Resmi SDN Citaleus II

Website resmi Sekolah Dasar Negeri Citaleus II.  
Menyajikan informasi profil sekolah, visi dan misi, data guru dan tenaga kependidikan (GTK), galeri kegiatan, serta informasi dan pengumuman sekolah.

## Tentang Website
Website ini dibangun sebagai media informasi resmi sekolah dan dikelola oleh operator sekolah SDN Citaleus II.

## Fitur Utama
- Profil dan Identitas Sekolah
- Visi dan Misi Sekolah
- Struktur Organisasi
- Data Guru dan Tenaga Kependidikan (GTK)
- Galeri Kegiatan Sekolah
- Berita dan Informasi Sekolah
- Kontak Sekolah
